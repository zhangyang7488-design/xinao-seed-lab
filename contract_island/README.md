# Grok 4.5 私人合同岛（低成本隔离）

## 原则（用户 2026-07-10）

**目标形状 ≈ 你和 Codex 的关系，但不必完全一样。**

| | 共用（像和 Codex） | 必须分家 |
|--|-------------------|----------|
| 工具 / skills / 镜像 / 证据大院 | 可以 | — |
| **角色** | — | **必须分** |
| **背后支撑链**（lane、检查点、合同岛、默认任务、完成叙事） | — | **必须分** |

- 同一 Grok 产品很多可复用，**不整仓 fork**  
- **角色 + 支撑链** 私人分家，防双向误会  
- **不对称**：4.5 可动 Admin；Admin 默认不能动 4.5
